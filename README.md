# apify-challenge

## how to use

### 1. <b> clone the repo </b>

```bash
git clone git@github.com:nickagliano/final_apify_challenge.git
```

### 2. <b>navigate to project and download npm dependencies</b>

```bash
npm install apify --save
npm install moment
npm install rimraf
```

### 3. <b>execute main.js</b>
 
```bash
node main.js
```
